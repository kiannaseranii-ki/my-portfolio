# Personal Portfolio Website

This is my personal portfolio website built to showcase my skills, projects, and provide a way to contact me directly.

The project is designed to be simple, fast, and accessible, while using real-world front-end workflows such as GitHub version control and Netlify deployment.

## Features

- Responsive personal portfolio layout
- Sections for skills, projects, and about me
- Contact form integrated with Netlify Forms
- Email notifications on form submission
- Deployed with continuous deployment via GitHub + Netlify

---

## Tech Stack

- HTML5 (Semantic markup)
- CSS3 (Responsive design)
- JavaScript (ES6+)
- React (used for interactive sections)
- Git & GitHub
- Netlify (Hosting & Forms)

---

## Contact Form

The contact form is connected to Netlify Forms and allows visitors to send messages without requiring a custom backend.

Messages are delivered directly via email and stored in the Netlify dashboard.

---

## Deployment

The project is deployed using Netlify with automatic deployments enabled:

- Any push to the `main` branch triggers a new deployment
- Hosting is fully static with no server-side code
